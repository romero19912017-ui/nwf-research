# Теория нейровесовых полей (NWF) — введение

Этот документ даёт минимальное введение в теорию NWF для тех, кто впервые с ней сталкивается.

---

## Проблема

При инкрементальном обучении (добавление новых классов) нейросети «забывают» старые — **катастрофическое забывание**. Методы вроде EWC или iCaRL частично решают проблему, но требуют переобучения и хранения exemplars.

**Идея NWF:** не переобучать модель, а **добавлять новые классы в индекс** без изменения весов. Для этого нужно представление, которое естественно поддерживает добавление.

---

## Ключевая концепция: заряд

Вместо точки (вектора) в латентном пространстве NWF использует **заряд** — пару (z, Sigma):

- **z** — центр (где объект «находится» в пространстве смыслов)
- **Sigma** — ковариация (насколько мы уверены в этом положении)

VAE даёт и то, и другое: encoder выдаёт (mu, log_var), откуда z = mu, Sigma = diag(exp(log_var)).

---

## Почему «поле»?

Потенциал в точке r от заряда i:

```
phi_i(r) = exp( - (r - z_i)^T Sigma_i^{-1} (r - z_i) / 2 )
```

Сумма по всем зарядам даёт «поле» потенциала в латентном пространстве. Запрос с высоким потенциалом — «внутри» облака обученных данных; с низким — out-of-distribution.

---

## Расстояние Махаланобиса

Обычное евклидово расстояние не учитывает, что в одних направлениях вариативность выше. Махаланобис масштабирует:

```
d² = (z_q - z_i)^T Sigma^{-1} (z_q - z_i)
```

Для диагональной Sigma: d² = sum_d (z_q_d - z_i_d)² / sigma_d².

**Симметричная метрика:** Sigma_comb = Sigma_q + Sigma_i — учитывает неопределённости и запроса, и индекса.

---

## Kalman и онлайн-обновление

Фильтр Калмана позволяет **уточнять** (z, Sigma) по новому наблюдению без переобучения. Применяется при дрейфе (например, объект поворачивается — новые кадры уточняют его представление).
