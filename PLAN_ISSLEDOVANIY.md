# План исследований NWF: от фундамента к прорыву

**Источник теории:** Препринт «Нейровесовые поля: теория семантического континуума»  
**Цель:** Проверить преимущества NWF, задать новое направление для ИИ.

---

## Польза NWF для общества

| Проблема | Решение NWF |
|----------|-------------|
| **Чёрный ящик** — нейросети не говорят «не уверен» | Σ — встроенный измеритель уверенности. Основа **доверенного ИИ** |
| **Экология обучения** — LLM = сжечь уголь города | Инкрементальность: новые заряды без переобучения. Путь к **зелёному ИИ** |
| **Проклятие контекста** — RAG = шпаргалка | Истинная долговременная память через суперпозицию полей. **RAG 2.0** |

## 5 магистральных направлений

## Направление 1: Фундамент — корректная реализация NWF

**Вопросы:**
- Влияние архитектуры H (VAE, NF, Diffusion) на z и Σ?
- Инициализация байесовского вывода предобученными энкодерами (CLIP, BERT)?
- Аппроксимация Σ: диагональ, низкий ранг, матрица точности?

**Эксперименты:**
1. **BayesianEncoder** — MAP-оценка для каждого объекта. VAE на MNIST/CIFAR.
2. **Валидация поиска:** Precision@10 при евклидовом расстоянии vs Махаланобис (Σq) vs симметричном (Σq+Σᵢ).
3. **Compression ratio** — NWF vs FAISS при одинаковой точности. Ожидание: NWF 152x vs FAISS 100x.

**Выход:** `nwf/vae_encoder.py`, `nwf/inference.py`, `experiments/01_fundament/`

---

## Направление 2: Инкрементальность и continual learning

**Вопросы:**
- NWF vs EWC, iCaRL, GEM?
- Дрейф концепций — можно ли обновлять заряды без переобучения?
- Влияние порядка классов? (NWF: порядок не должен влиять)

**Эксперименты:**
1. **Split-MNIST / Split-CIFAR-100** — сравнение NWF (фиксированный VAE) с EWC, iCaRL. Метрика: точность на всех классах после последней задачи.
2. **Дрейф концепций** — синтетический датасет (например, «мода» меняется во времени). Обновление зарядов через Kalman без переобучения.

**Выход:** `experiments/02_incremental/`, `experiments/02_drift/`

---

## Направление 3: NWF как память для LLM (RAG 2.0)

**Вопросы:**
- Интеграция NWF-хранилища в пайплайн генерации?
- Семантическая арифметика: z(Париж) + z(столица) → z(Франция)?
- Детекция галлюцинаций: низкая плотность поля → «я не знаю»?

**Эксперименты:**
1. **NWF-RAG** — SQuAD / Natural Questions. Документы → заряды (E5/BERT + байесовский вывод). Поиск по Махаланобису. Сравнение с FAISS+RAG по точности и полноте.
2. **Семантическая интерполяция** — факты о городах. z(Рим) и z(Франция) → близость к z(Италия)?
3. **OOD-детекция** — запросы вне базы. Корреляция Φ(zq) и «ответ есть в базе».

**Выход:** `experiments/03_rag/`, `experiments/03_ood/`

---

## Направление 4: Доверенный ИИ — интерпретируемость и калибровка

**Вопросы:**
- Интерпретация Σ: «неуверен из‑за противоречивых примеров»?
- Карта ландшафта Φ(r): «горы» = уверенные знания, «низины» = противоречия?
- Калибровка вероятностей из голосования соседей?

**Эксперименты:**
1. **Анализ Σ** — зашумленные объекты → рост Σ. Отсев «мусорных» запросов.
2. **Визуализация поля** — t-SNE/UMAP + контуры Φ(r). Границы классов = низины.
3. **Reliability diagram** — NWF vs Softmax нейросети. Ожидание: NWF лучше калиброван на OOD.

**Выход:** `experiments/04_trusted_ai/`

## Направление 5: Kalman-NWF и хроно-поля (T-NWF) [РЕАЛИЗОВАНО]

**Фильтр Калмана — неотъемлемая часть теории NWF** (Аксиома А5). Рекурсивное обновление (z, Sigma) по принципу свободной энергии.

**Теоретическое обоснование:** Baltieri & Isomura (2021) — Kalman = steady-state градиентного спуска по свободной энергии.

**Реализация:** `nwf/kalman.py` — KalmanEncoder с encode_static(), update().

**Архитектура Kalman-кодировщика:**
```
1. Инициализация: z₀, Σ₀ = encoder(D)
2. Итерации t=1..T:
   - Предсказание: z_t|t-1 = F·z_{t-1}, Σ_t|t-1 = F·Σ·Fᵀ + Q
   - Наблюдение: y_t = decoder(z_t), R_t = шум
   - Обновление: K = Σ·Hᵀ·(H·Σ·Hᵀ + R)⁻¹, z_t = z + K·(D - y), Σ_t = (I - K·H)·Σ
3. Выход: (z_T, Σ_T)
```

**Эксперименты Kalman-NWF:**
1. **Скорость кодирования** — Kalman vs градиентный спуск vs Adam. Ожидание: 5–10x меньше итераций.
2. **Онлайн-обновление** — дрейф концепций. Обновление заряда через Kalman при новых данных.
3. **Калибровка уверенности** — корреляция trace(Σ) и ошибки. Reliability diagram.
4. **Шумоподавление** — зашумлённое D → Kalman → восстановление. PSNR, SSIM vs VAE, BM3D.
5. **T-NWF (темпоральные поля)** — z(t) для временных рядов. Предсказание z(t+1). Визуализация семантических траекторий.

**Выход:** `nwf/kalman_encoder.py`, `experiments/05_kalman/`, `experiments/05_tnwf/`
