# План исследований NWF

**Источник теории:** Препринт «Нейровесовые поля: теория семантического континуума» (раздел 3–6)

## Фаза 1: Корректная реализация (этапы 1–3)

### Этап 1.1. VAE и байесовское кодирование
**Цель:** Получить (z, Σ) для каждого объекта через MAP-оценку.

- [ ] VAE для MNIST (латентная размерность 64)
- [ ] Инференс: для каждого D → zᵢ, Σᵢ (диагональ)
- [ ] Оптимизация ELBO или MAP относительно z
- [ ] Σᵢ из гессиана или вариационной оценки

**Выход:** `nwf/vae_encoder.py`, `nwf/inference.py`

### Этап 1.2. Ядро NWF с Махаланобисом
**Цель:** Charge(z, Σ), потенциал, расстояние.

- [ ] Класс Charge с z и Σ (диагональ)
- [ ] Функция mahalanobis_dist(q, i) с учётом Σq и Σᵢ
- [ ] Потенциал φᵢ(r) = exp(−d²_M/2)
- [ ] Суперпозиция Φ(r) = Σ φᵢ(r)

**Выход:** `nwf/core.py`

### Этап 1.3. Хранилище и поиск
**Цель:** Индекс для k-NN по Махаланобису.

- [ ] NWFStorage: add(z, Σ, label), search(zq, Σq, k)
- [ ] Реализация через torch.cdist или собственная метрика
- [ ] Поиск по φq(zⱼ) или d_M

**Выход:** `nwf/storage.py`, `nwf/search.py`

## Фаза 2: Валидация гипотез (этапы 4–7)

### Этап 2.1. Эксперимент H2: Устойчивость к шуму
**Гипотеза:** Σ повышает устойчивость vs статические эмбеддинги.

- [ ] Протокол: шум σ = 0, 0.1, 0.2, 0.5 на запросах
- [ ] NWF (Махаланобис) vs FAISS/эмбеддинги (L2)
- [ ] Метрики: Precision@10, Recall@10
- [ ] Целевой результат: падение NWF < падение FAISS (по статье: −5% vs −12% при σ=0.2)

**Выход:** `experiments/02_noise/`, `results/noise_nwf_vs_faiss.csv`

### Этап 2.2. Эксперимент H1: Семантическое сжатие
**Гипотеза:** NWF сжимает лучше при той же точности поиска.

- [ ] Метрика: compression ratio = размер исходных данных / размер NWF (z + Σ)
- [ ] NWF vs FAISS (только z) на MNIST
- [ ] Precision@10, Recall@10
- [ ] Целевые ориентиры: NWF ~152x, FAISS ~100x (из статьи)

**Выход:** `experiments/01_compression/`, `results/compression.csv`

### Этап 2.3. Эксперимент H3: Инкрементальное обучение
**Гипотеза:** NWF не забывает при добавлении новых классов.

- [ ] Задачи: классы 0–2, 3–5, 6–9 (последовательно)
- [ ] NWF: добавление зарядов, VAE фиксирован
- [ ] EWC, iCaRL, наивное дообучение — те же разбиения
- [ ] Метрика: точность на всех виденных классах после каждой задачи

**Выход:** `experiments/03_incremental/`, `results/incremental_nwf_ewc_icarl.csv`

### Этап 2.4. Эксперимент: Семантическая интерполяция
**Цель:** Демонстрация z_new = α·zA + (1−α)·zB.

- [ ] Визуализация через декодер VAE
- [ ] Примеры для MNIST (цифры 3 и 5)

**Выход:** `experiments/04_interpolation/`, `results/interpolation_demo.png`
